A Form of Resistance

Welcome! This is a companion site for the workshop “A Form of Resistance: Manipulating Web Forms,” presented on May 9th, 2024. The code for this site and the script for the workshop can be found on GitHub.

All external links on this site open in a new tab.

Documentation for browser web development tools

  • Firefox
  • Google Chrome (and other Chromium-based browsers, e.g. Brave)
  • Microsoft Edge
  • Safari — Note that Safari’s developer tools must be enabled before they can be accessed.

Further reading, and resources

Ordered by whimsy.

  • “Exploits of a mom” comic. xkcd.
  • genders.wtf, a horrifying and funny collection of “gender” fields in forms.
  • “The gauntlet”, a page which takes the form fields collected in genders.wtf and loads them into an infinite-scroll web form.
  • “Tiny Internets dispatch 01: our internet”. Spencer Chang. 2022.
  • “The disenshittified internet starts with loyal "user agents"”. Cory Doctorow. 2024.
  • “When transparency and technology activists come together, contacting Congress gets easier and easier”. Electronic Frontier Foundation. 2014.
  • “Addressing delays in Democracy.io and the EFF Action Center message delivery”. Electronic Frontier Foundation. 2017.
  • Data Feminism. Catherine D’Ignazio and Lauren F. Klein. 2020. If you only have time to read a little, I recommend chapter 4, “What gets counted counts”.
  • “Falsehoods programmers believe about names”. Patrick McKenzie. 2010.
  • “Awesome falsehood”, an index of “falsehoods programmers believe in” lists and essays.
  • “Deceptive patterns”, a website which describes, collects, and monitors cases of “tricks used in websites and apps that make you do things that you didn't mean to, like buying or signing up for something.”
  • “Airlines to begin adding new gender option for ‘non-binary’ fliers”. USA Today. 2019.
  • “Introduction to HTML” tutorial. MDN Web Docs.
  • “Web forms — Working with user data” tutorial. MDN Web Docs.
  • “HTML elements — Forms”, an index with every HTML element available inside a form. MDN Web Docs.
